Fontutils

The fontutils package includes the programs bpltobzr, bzrto, charspace, fontconvert, gsrenderfont, imageto, imgrotate, limn, and xbfe. These create fonts for use with Ghostscript or TeX (starting with a scanned type image and converting the bitmaps to outlines), convert between font formats, etc.

The package also includes the libraries libbzr.a, libgf.a, libpbm.a, libpk.a, libtfm.a, and libwidgets.a.
