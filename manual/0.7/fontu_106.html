12.2 Invoking BPLtoBZR

This section describes the options that BPLtoBZR accepts. See section 3.3 Command-line options, for general option syntax.

`-help'
Print a usage message. See section 3.3.2 Common options.

`-output-file filename'
Output to filename (if it has a suffix) or to `filename.bzr' (if it doesn't).

`-range char1-char2'
Only output characters with codes between char1 and char2, inclusive. (See section 3.3.2 Common options, and 3.3.3 Specifying character codes.)

`-verbose'
Output progress reports.

`-version'
Print the version number of this program.
