8. Fontconvert

Fontconvert performs manipulations on bitmap fonts: conversion to other formats, merging multiple fonts, adjusting individual characters, moving characters around within a font, ...

The input is either a GF or a PK bitmap font, and in some circumstances, a TFM file. (See section 5.1 File format abbreviations.) The output varies according to the options specified.
